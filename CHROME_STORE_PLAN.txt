## PHASE 4: SUBMISSION PROCESS

### 4.1 Package Extension
- [ ] Create ZIP file of extension folder
- [ ] Include: manifest.json, all HTML/CSS/JS files, icons
- [ ] Exclude: node_modules, .git, development files
- [ ] Maximum size: 128MB (our extension is <1MB)

### 4.2 Upload to Chrome Web Store
1. Go to Chrome Web Store Developer Dashboard
2. Click "New Item"
3. Upload ZIP file
4. Fill out store listing:
   - Title: "Em Dash Optimizer"
   - Category: "Productivity"
   - Language: English
   - Visibility: Public

### 4.3 Complete Store Listing
- [ ] Upload all screenshots
- [ ] Add store tiles
- [ ] Set privacy policy URL
- [ ] Review all information
- [ ] Save as draft

### 4.4 Submit for Review
- [ ] Click "Submit for Review"
- [ ] Wait for Google review (1-7 days typical)
- [ ] Respond to any feedback if rejected
